> 发布于 2026-03-28

网站终于搭建起来了！

## 这个网站用了什么技术

- **Obsidian** — 本地 Markdown 编辑与知识管理
- **VitePress** — 将 Markdown 转换为静态网站
- **GitHub** — 代码托管与版本管理
- **Cloudflare Pages** — 免费自动化部署

## 工作流程

每次只需要在 Obsidian 中写 Markdown，提交到 GitHub，网站就会自动更新。

```
写 Markdown → git push → Cloudflare 自动部署 → 网站更新
```
